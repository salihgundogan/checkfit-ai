# 🧠 CheckFit AI - Akıllı Diyet Asistanı

CheckFit AI, kullanıcı geri bildirimleriyle sürekli öğrenen (HITL - Human in the Loop) yapay zekâ tabanlı bir mobil sağlık uygulamasıdır.

Bu proje, TÜBİTAK 2209-A kapsamında geliştirilmektedir.

---

## 🚀 Proje Amacı

- Görüntü işleme ile yemek tanıma
- Kişiye özel diyet önerileri
- Kullanıcı geri bildirimleriyle kendini geliştiren AI sistemi

---

## 🧠 Özellikler

- 📸 Kamera ile yemek tanıma
- 🔁 Human-in-the-loop öğrenme
- 📊 Kalori ve besin analizi
- 🎯 Kişiselleştirilmiş diyet önerileri
- 🧠 Davranışsal geri bildirim sistemi (Skinner)

---

## 🏗️ Teknolojiler

- React Native
- Python
- TensorFlow / PyTorch
- Flask / FastAPI
- Supabase
